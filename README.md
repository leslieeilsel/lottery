<h1 align="center">大乐透预测</h1>


#### Usage

- 同步开奖历史数据

  ```shell
  php artisan sync:history {pageSize=30}
  ```


- 生成幸运球

  ```shell
  php artisan luckyballs {count=1} {draw?}
  ```


- 计算幸运球中奖规则、中奖金额

  ```shell
  php artisan calculate:lucky
  ```
